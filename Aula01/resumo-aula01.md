# Aula 01 - Lógica de Programação

## O que é algoritimo
Algoritimo é uma sequência de passos usados para resolver um problema ou execultar uma tarefa.

## Entrada, processamento e saída

Entrada: são os dados que o algoritimo reccebe.

Porcessamento: é o que o algortimo faz com os dados.

Saída: É o resultado produzido.

## Exemplo do dia a dia

Escovar os dentes:

1. Pegar a escova
2. Colocar pasta de dente
3. Escovar os dentes
4. Enxaguar a boca

## Palavras-chave da aula

Algoritmo – sequência de passos para resolver um problema.

Entrada – dados recebidos pelo algoritmo.

Processamento – transformação dos dados.

Saída – resultado final.

Lógica – forma organizada de pensar para resolver problemas.
